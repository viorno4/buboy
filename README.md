# BUBOY!
An interactive TUI PyGame where you buy stuff from Filipino "sari-sari" stores.

### Dependencies
* Python 3

### How to play
You are playing as Buboy, a quirky Filipino kid in an average Filipino household. You are given a shopping list by your mother. You are tasked to buy everything in this shopping list from nearby sari-sari stores.

Not all sari-sari stores have everything in stock. Prices also vary across stores, so spend your money wisely! Of course, you can ask for "tawad" (discount) if you feel lucky. Let's just hope the "bantay ng tindahan" is in a good mood to give you a discount:)

### Installation
This repo is intended to be shared with my beautiful girlfriend, Armie, who understandably doesn't know how to code (it's confusing, I totally get her). So, excuse me for putting up a baby-step installation guide for this program.

Though frankly, I assume that you already have Python and git installed. If you are my girlfriend and you find this bit confusing, you always have the luxury to hop into my DM :))

1. Clone into this repository
Paste the following code into your terminal. Wherever you run this code is wherever you'll install this game in.
`git clone https://github.com/viorno4/buboy`

2. `cd` into the buboy folder.
`cd buboy`

3. Run with python
Type or paste this code into your terminal to run the game.
`python3 main.py`
If that doesn't work, try this:
`py main.py`

Thank you for playing!
